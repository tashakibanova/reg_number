# Track & Send — Рассылка трек-номеров

Статическое веб-приложение для массовой рассылки трек-номеров посылок через сервис UniSender. Работает полностью в браузере, не требует бэкенда.

---

## Возможности

- Таблица получателей с ручным вводом и импортом CSV
- Редактор HTML-шаблона письма с живым предпросмотром
- Массовая отправка через API UniSender с прогресс-баром
- История отправок в localStorage

---

## 1. Регистрация на UniSender и получение API-ключа

1. Перейдите на [unisender.com](https://www.unisender.com) и зарегистрируйтесь (бесплатный план — до 1500 писем в месяц).
2. Войдите в личный кабинет.
3. Перейдите в **Настройки → Интеграции → API**.
4. Скопируйте ваш **API-ключ**.
5. Добавьте и **верифицируйте** email-адрес отправителя в разделе **Настройки → Отправители → Email-адреса**.

---

## 2. Настройка переменных окружения

Скопируйте файл `.env.example` в `.env.local`:

```bash
cp .env.example .env.local
```

Откройте `.env.local` и заполните все поля:

```env
VITE_ADMIN_PASSWORD=ваш_надёжный_пароль
VITE_UNISENDER_API_KEY=ваш_api_ключ
VITE_SENDER_EMAIL=noreply@example.com
VITE_SENDER_NAME=Служба доставки
```

> Никогда не добавляйте `.env.local` в git — файл уже исключён через `.gitignore`.

---

## 3. Запуск локально

```bash
npm install
npm run dev
```

Приложение откроется по адресу http://localhost:5173/reg-number/

---

## 4. Деплой на GitHub Pages

### Первичная настройка репозитория

1. Создайте репозиторий на GitHub с именем `reg-number`.
2. Привяжите его к локальной папке:

```bash
git remote add origin https://github.com/ВАШ_ЛОГИН/reg-number.git
git push -u origin main
```

### Деплой

```bash
npm run deploy
```

Команда автоматически:
1. Соберёт проект (`npm run build`)
2. Опубликует папку `dist` в ветку `gh-pages`

После этого в настройках репозитория GitHub перейдите в **Settings → Pages** и убедитесь, что источник — ветка `gh-pages`.

Приложение будет доступно по адресу:
```
https://ВАШ_ЛОГИН.github.io/reg-number/
```

## Формат CSV для импорта

Файл должен содержать по одной записи на строку. Разделитель — запятая или точка с запятой:

```
RA123456789RU,ivan@example.com
RA987654321RU;maria@example.com
```
